# G_xy Library


### วิธีติดตั้ง
```
pip install G_xy==0.0.x
```
### วิธีใช้งาน

```
from G_xy import G_xy
```
### การเปลี่ยนค่าตัวเเปร
```
x = G_xy()    
x.g1 = 9   
x.t = 0.01
x.V_initial3 = 6
```
### ดูตัวอย่างโค้ด
```
print(G_xy()) 
```
### เเสดงกราฟทั้งหมดพร้อมกัน
```
x.xy()
```
### เเสดงกราฟเเยกที่ละกราฟ
```
x.xt()
x.yt()
x.vxt()
x.vyt()
```
